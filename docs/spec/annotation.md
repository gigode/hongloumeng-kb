# 标注规范

首版标注只追求高价值实体，不做全量词法标注。所有标注必须能去标记还原正文。

| 标记 | 类型 | 说明 |
|------|------|------|
| `〖@人物〗` | person | 小说人物、神仙僧道、重要称谓归一到人物 |
| `〖=地点〗` | place | 城市、府邸、园林、梦境地点 |
| `〖&府系〗` | household | 贾家、甄家、荣国府、宁国府等家族或府系 |
| `〖#身份〗` | role | 丫鬟、姑娘、公子、知府等身份角色 |
| `〖•物件〗` | object | 通灵宝玉、金锁、风月宝鉴等叙事物件 |
| `〖{诗文〗` | text | 判词、曲文、书名、诗文单元 |
| `〖_概念〗` | concept | 梦、情、空、色、薄命、因缘等母题 |
| `〖%时间〗` | time | 明确时间词 |

别名用 `显示名|规范名`：

```markdown
〖@宝玉|贾宝玉〗
〖@黛玉|林黛玉〗
```

原则：

- 不在同一处嵌套标注。
- 优先标注会参与关系或索引的实体。
- 模糊解释性的关系暂不标，留给后续人工审查。
- 自动标注结果必须通过完整性测试：去标记后与 raw 正文一致。
